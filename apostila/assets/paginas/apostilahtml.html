Visão Geral sobre HTML

A sigla HTML significa "HyperText Markup Language" (Linguagem de marcação de hipertexto).
Ele é responsavel por organizar e dar significado a uma página web, pois, sem ele, os navegadores não saberiam como exibir um simples parágrafo, imagem ou qualquer outro tipo de conteúdo para o usuario.
Podemos dizer que o HTML seria como o nosso esqueleto, definindo o que é cada parte do "corpo" usando as tag's.
