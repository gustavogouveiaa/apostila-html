O que é HTML?

Siglapara HyperText Markup Language (Tradução para PT-BR, Linguagem de marcação de hipertesto), O HTML é responsável pela estruturação de um website. Esta é a única linguagem que os navegadores atuais conseguem entender e enterpretar para o usuário.

Ela será usada para determinar cada elemento presente na página a ser criada, como por exemplo parágrafos usando a tag <p>, uma lista não-ordenada usando a tag <ul>, entre várias outras.
