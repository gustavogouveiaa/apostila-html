Tipos de listas

Existem três tipos que podemos usar: Lista ordenada, lista não-ordenada e lista de definição

Lista não-ordenada

A tag para essa lista é a "<ul> </ul>", dentro dela temos que usar a tag para adicionar os itens da lista "<li> </li>". Como diz o nome, aqui a ordem dos itens não tem importancia alguma.

  • Item 1
  • Item 2
  • Item 3

Lista ordenada

A tag para essa lista é a "<ol> </ol>", dentro dela temos que usar a tag para adicionar os itens da lista "<li> </li>". Ao contrario da lista a cima, essa lista agora a sequência dos itens tem importancia.

  1. Item 1
  2. Item 2
  3. Item 3

Usando o CSS ou o atributo "style", podemos cubtituir a ordem numérica pela ordem alfabética usando a propriedade "list-style-type: upper-alpha;"

  2. Item 2
  3. Item 3
