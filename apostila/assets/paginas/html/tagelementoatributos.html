Elemento HTML

Tags e conteúdo

Um elemento é composto por duas partes principais: Tag de abertura/fechamento e o conteúdo.

A tag representada pela letra "p" irá indicar ao browser de que aquela linha será definida como um parágrafo. Tudo o que estiver entre a tag de abertura e fechamento é o conteúdo, que por sua vez, sera interpretado como paragrafo. Assim, teremos um ELEMENTO HTML completo!
Existem alguns elementos que possuem apenas a tag de abertura, não necessitando de fechamento, como, por exemplo, a tag <img >, usada para inserir uma imagem ao documento.

Atributos HTML

Os atributos são valores adicionais que podem configurar ou ajustar os elementos

Alguns atributos mais usado são:

  • class - Usado para nomear um elemento. Frequentemente usado junto ao CSS
  • href - Usado para direcionar o usuário a um determinado site ou local do próprio documento.
  • id - Usado para dar um nome Unico a um elemento. Frequentemente usado junto ao CSS
